🎯 Что это за программа?
Это веб-мессенджер в стиле аниме "Токийский Гуль" (Tokyo Ghoul). Полноценное приложение для общения, которое работает прямо в браузере, не требуя установки.

🛠️ Что делает этот код?
1. Система аутентификации
Регистрация новых пользователей (email + пароль)

Вход в существующий аккаунт

Все данные сохраняются локально в браузере

2. Управление профилем
Настройка имени, username (@никнейм), телефона

Загрузка/удаление аватара (фото профиля)

Все изменения сохраняются

3. Система чатов и контактов
Список контактов с аватарами и онлайн-статусом

Переключение между чатами через вкладки

Поиск и добавление новых друзей

4. Обмен сообщениями
Отправка текстовых сообщений

Получение автоматических ответов от персонажей

История сообщений сохраняется

Отметки времени на сообщениях


6. Стилизация
Темная тема в черных тонах (стиль аниме)

Анимации появления сообщений

Адаптивный дизайн

Иконки-аватары персонажей

👥 Для чего это нужно людям?
1. Обучение и демонстрация
Показывает, как создать полноценное веб-приложение

Демонстрирует работу с HTML, CSS и JavaScript

Пример локального хранения данных

2. Фанатам аниме
Общение с любимыми персонажами

Погружение в атмосферу "Токийского Гуля"

Тематическое оформление

3. Прототипирование
База для создания реального мессенджера

Можно доработать под свои нужды

Добавить реальный сервер вместо локального хранения

4. Развлечение
Интерактивное общение с чат-ботами
